## Updating the chart

 1. Follow the instructions in `Config.hs`
 2. Make sure that the `colgroup`s in `mkTable` in `Layout.hs` cover all
    columns.
 3. Make sure that `.first-column` and `.other-columns` in `css/custom.css` add
    up to 100%.
